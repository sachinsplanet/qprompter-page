What's inside

Eight tools. One popup. Zero babysitting.

Every feature built for the kind of AI work that takes more than one message to get right.

Prompt Queue Engine

Add, reorder, edit and run multiple prompts sequentially with configurable gap delays between dispatches.

Background Execution

Queue keeps running after the popup closes. Triple keep-alive beats Chrome's 30-second kill.

Smart Response Waiter

Uses MutationObservers to detect when the AI actually finishes streaming. No timers, no guessing.

Image Mode

Rewrites prompts with structural image keywords and genre presets: Product Shot, 3D Render, Anime.

Prompt Library

100+ categorized templates for Coding, Marketing, Business, and Image Generation with dynamic variables.

Locked Tab Binding

Extension locks to the exact tab it started on. Prompts never dispatch to another tab mid-run.

History Log

Full run history with per-prompt status, error banners, and timestamps. Know exactly what ran and when.

Backup & Restore

Export your entire config and queue as JSON. Import it back in one click. Your workflow, your ownership.

Locked tab model means prompts dispatch to exactly one tab — the one you started on. No multi-tab clashes.

Self-healing queue state persists through browser restarts. Close Chrome, reopen — your queue picks up where it left off.
